Issues

This page covers things that might cause problems or unexpected behavior in the A+ Grass asset.

"Grass Is Darker Than Expected"

It could happen on the package import when first trying the asset due to compatibility of all Rendering Paths and not being assigned to the correct one on import. Just go to your project's Settings folder and on your Renderer Data > Rendering, swap Rendering Path to a different one and back and it should fix it.

Otherwise, make sure that your material setup is correct, check your Texture Ramp, Colour, or Colour Variant and if you're not just overriding your base colour with another.

"Wind Effect Is Broken"

Issues will most likely occur if you are using a custom model/mesh and its not setup correctly.

Check following:

  • The original/base scale needs to be 1,1,1 (Anything else like 100,100,100 scale will cause issues) Ensure you export your mesh correctly and you apply normal scale.

  • Could be as simple as the values not set and the effect being disabled. Check the material and the values under the wind section and if you have assigned a noise texture.

"Base Texture Is Not Displaying Correctly"

Issues will most likely occur if you are using a custom model/mesh and its not setup correctly.

Check following:

  • Ensure the UV is done correctly. The mesh face should cover the full box as that is how the texture will be mapped. (Check its rotate properly as well)

  • Check your tiling and offset values on the base texture. You can reset them using R button to reset to default values.
