Getting Started

Welcome to the A+ URP Grass asset! This documentation will help you get started quickly and efficiently.

You can import the A+ URP Grass asset into your Unity project by downloading it from the Unity Asset Store or from the Package Manager directly in your project.

In the folder of the asset you will find a Demo Scene which you can use to explore the features and functionality of the A+ URP Grass asset on your own. Otherwise you can follow along with the documentation to learn how to set up and use the asset in your own project.

Grass Setup

The package comes with ready prefabs that you can drag and drop into your scene.
