Changelog

Documentation of the version history and release notes for the A+ URP Grass asset.

Version 1.1.0

22/06/2026

A few fixes and additions / Distance Fading feature.

Fixes:

  • Version 6000.3+ material rendering fix (correction in Deferred path)
  • Added Area Size variable in Grass Manager for correct functionality between mutliple interactors in a larger radius

Additions:

  • Grass/Mesh Bend Pivot Point variable
  • Perspective Correction Height Start variable (extra bend pivot adjustment)
  • Distance Fading feature (smooth + dither fade options)

Version 1.0.0

23/05/2026

Initial release of the A+ URP Grass asset.

Supported for Unity Versions:

  • 6000.3+
  • 6000.0+
  • 2022+

Contents:

  • Grass Shader
  • Core Scripts
  • Essential Textures
  • Ready To Go Prefabs
  • Example Scenes
  • Documentation

Features:

  • Texture based grass rendering
  • Adjustable color with gradient settings
  • Color variation setup with noise map
  • Character and mouse interactive behavior
  • Bent grass colour trail tint
  • Wind noise with 2 blend modes
  • Perspective correction
  • Adjustable runtime weather
